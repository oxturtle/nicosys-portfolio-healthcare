# Healthcare Data Analytics Portfolio

## Overview
This repository showcases my work in healthcare data analytics, focusing on practical, real‑world data skills used to explore, clean, analyze, and visualize healthcare datasets. The goal is to demonstrate strong analytical thinking, technical proficiency, and clear communication of insights.

## Tools & Technologies
I use a combination of industry‑standard tools to manage, analyze, and visualize data:

- **Excel** – data exploration, cleaning, and quick analysis  
- **PostgreSQL** – database management, querying, and transformation  
- **Power BI** – interactive dashboards and visual storytelling  
- **Git Bash** – command‑line navigation, version control, and repository management  
- **VS Code** – coding, documentation, and project organization  

## Repository Structure
~~~
data/
├── raw/        # Original source data
├── cleaned/    # Processed and analysis-ready datasets
sql/            # PostgreSQL queries and transformations
scripts/        # Utility scripts (future expansion)
reports/        # Power BI files, summaries, and insights
~~~


Each folder is designed to mirror a real analytics workflow:  
**collect → clean → analyze → visualize → document.**

## Purpose
This portfolio demonstrates my ability to:

- Work with healthcare datasets  
- Clean and transform data using SQL and Excel  
- Build meaningful dashboards in Power BI  
- Apply version control using Git and Git Bash  
- Organize projects using VS Code and GitHub  
- Communicate insights clearly and professionally
